# Evidence Rules

## Evidence Table Fields

Each finding should record:

- Document.
- Section or note.
- Reported metric.
- Comparison metric.
- Calculation.
- Factual observation.
- Interpretation.
- Possible benign explanation.
- Confidence.
- Missing evidence.

## Evidence Strength

Use Weak when evidence comes from a single metric, incomplete data, or one quarter only.

Use Moderate when at least two statements or periods support the observation.

Use Strong when multiple periods, multiple statements, notes, and management commentary all support the same concern.

## Calculation Rules

Show formulas. State whether the calculation uses quarterly, year-to-date, or trailing data. Avoid mixing periods unless the output clearly explains the normalization.

## Missing Evidence

Explicitly list missing schedules, prior periods, note disclosures, segment detail, reconciliations, or cash-flow information that would improve confidence.
